## 项目介绍
  这个项目实现了四则运算，这个项目不仅实现了简单的加减乘除，还支持二元一次方程，开根号，并且可以判断计算结果的正确性等等；这个项目的设计主要是为了方便老师及其他需要出题的人员，同时也给想要学习四则运算的同学以及其他人员提供了极大的便利，最终这个项目不仅可以做成手机APP应用程序，还可以做成台式机上的服务器模式。
